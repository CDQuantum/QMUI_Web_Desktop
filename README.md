# QMUI Web Desktop
> 基于 [QMUI Web](https://github.com/QMUI/qmui_web) 的桌面 App，它可以管理基于 QMUI Web 进行开发的项目，并提供了编译提醒，出错提醒，进程关闭提醒等额外的功能。 

详细介绍及文档请浏览：[QMUI Web 官网](http://qmuiteam.com/web)
